# 🧠 Mental Health EDA & Prediction
A data-driven project focused on **Exploratory Data Analysis (EDA)** and **Machine Learning modeling** to understand and predict mental health conditions based on various personal and workplace-related factors.
## 📌 Project Overview
Mental health is a critical aspect of overall well-being, yet it is often overlooked.
This project aims to:
* Analyze patterns and trends in mental health data
* Identify key factors affecting mental health
* Build predictive models to classify mental health conditions
## 📊 Dataset Description
The dataset contains features such as:
* Age
* Gender
* Work environment
* Family history
* Treatment history
* Mental health condition (target variable)
## 🔍 Exploratory Data Analysis (EDA)
Performed detailed analysis to uncover insights:
* Handling missing values
* Data cleaning and preprocessing
* Univariate and bivariate analysis
* Correlation heatmaps
* Distribution plots
### 📈 Key Insights
* Work stress and environment strongly impact mental health
* Family history plays a significant role
* Certain age groups are more prone to mental health issue
## ⚙️ Workflow
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Engineering
5. Model Building
6. Model Evaluation
